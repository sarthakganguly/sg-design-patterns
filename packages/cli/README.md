# @sg-pattern-engine/cli

Command-line interface for PatternEngine.

## Install

\`\`\`bash
npm install -g @sg-pattern-engine/cli
\`\`\`

## Usage

\`\`\`bash
# Generate with defaults (1024x1024 Perlin, random seed)
pattern-engine generate

# Generate from a config file
pattern-engine generate --config ./my-pattern.json

# Explicit arguments
pattern-engine generate --width 2048 --height 2048 --output result.png
\`\`\`
